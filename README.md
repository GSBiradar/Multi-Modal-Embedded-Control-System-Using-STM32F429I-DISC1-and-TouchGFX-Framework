# Multi-Modal-Embedded-Control-System-Using-STM32F429I-DISC1-and-TouchGFX-Framework
Advanced embedded HMI system controlling STM32F429I-DISC1 on-board LEDs via TouchGFX GUI. Features touch-screen buttons for LED13/LED14 control and joystick-operated virtual LED visualization. Complete embedded GUI implementation with hardware integration.
